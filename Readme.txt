 University Management System

A Java Swing-based desktop application for managing student records in a university. The system allows users to add, view, and delete student information, calculate GPA and grades automatically, generate student reports, and export data to a text file.

 Features
- Add and manage student records
- Automatic GPA and grade calculation
- Generate student reports
- Export data to `students.txt`
- User-friendly GUI
- OOP concepts (Abstraction, Inheritance, Interface, Polymorphism)

Technologies Used
- Java
- Swing & AWT
- File Handling
- Object-Oriented Programming (OOP)
